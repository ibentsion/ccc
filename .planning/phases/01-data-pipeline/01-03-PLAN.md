  <behavior>
    `FIMDataset(records: list[dict], stage_config: StageConfig, seed: int)`:
    - Pre-generates FIM samples during `__init__` by calling `create_fim_sample(r["input"], n, seed)` for each record `r`
    - `n` is sampled uniformly from `range(stage_config.min_lines, stage_config.max_lines + 1)` using `random.Random(seed + i)` for record index `i` — ensures reproducibility while varying n across samples
    - Records where `create_fim_sample` raises `GapError` are silently skipped
    - `__len__` returns number of successfully generated samples
    - `__getitem__(i)` returns the sample dict (including `fim_text`, `prefix`, `suffix`, `middle`, `n_lines`)

    `HybridReplayDataset(current: FIMDataset, prior: list[FIMDataset], replay_ratio: float = 0.25)`:
    - For Stage 1 (no prior stages): `prior=[]`, behaves as pure `current` dataset — replay_ratio has no effect
    - For Stage N (prior stages exist): each `__getitem__` call returns from `current` with probability `(1 - replay_ratio)` or from a random prior-stage dataset with probability `replay_ratio`
    - Selection uses index-based deterministic routing: `idx % 4 != 0` → current, `idx % 4 == 0` → prior (achieves ~75/25 without runtime randomness, making batches deterministic)
    - When routing to prior, select prior stage uniformly: `prior[idx % len(prior)]`, then sample from that dataset: `prior_ds[idx % len(prior_ds)]`
    - `__len__` returns `len(current)` — DataLoader length is determined by current stage size

    Test cases:
    - `test_fim_dataset_generates_samples`: FIMDataset with 10 records and Stage 1 config → len > 0
    - `test_fim_dataset_items_have_fim_text`: every item has `fim_text` starting with `<｜fim▁begin｜>`
    - `test_fim_dataset_skips_gap_errors`: records with trivial code (only `def foo(): pass`) are skipped silently
    - `test_hybrid_stage1_no_replay`: HybridReplayDataset with `prior=[]` — all items come from current stage (check `n_lines == stage_config.min_lines`)
    - `test_hybrid_replay_ratio`: HybridReplayDataset with current (n=2) and prior=[stage1_ds] — check that roughly 25% of items have `n_lines == 1` across 100 items
    - `test_hybrid_deterministic`: same idx returns same item across two instantiations
  </behavior>

2. `class FIMDataset(Dataset)`:
   ```python
   def __init__(self, records: list[dict], stage_config: StageConfig, seed: int = 42):
       self.samples = []
       rng = random.Random(seed)
       line_choices = list(range(stage_config.min_lines, stage_config.max_lines + 1))
       for i, record in enumerate(records):
           n = rng.choice(line_choices)
           try:
               sample = create_fim_sample(record["input"], n=n, seed=seed + i)
               self.samples.append(sample)
           except GapError:
               pass  # silently skip records with no valid gap
   ```
   `__len__`: return `len(self.samples)`
   `__getitem__`: return `self.samples[idx]`

3. `class HybridReplayDataset(Dataset)`:
   ```python
   def __init__(self, current: FIMDataset, prior: list[FIMDataset], replay_ratio: float = 0.25):
       self.current = current
       self.prior = prior
       self.replay_ratio = replay_ratio
       # Pre-compute routing: index i routes to prior if i % 4 == 0 AND prior is non-empty
       # This gives 1/4 = 25% replay deterministically

   def __len__(self):
       return len(self.current)

   def __getitem__(self, idx):
       if self.prior and idx % 4 == 0:
           # Route to prior stage: pick prior dataset round-robin, then sample from it
           prior_ds = self.prior[idx % len(self.prior)]
           prior_idx = idx % len(prior_ds)
           return prior_ds[prior_idx]
       return self.current[idx % len(self.current)]
   ```

```python
from typing import Optional
import pytorch_lightning as pl
from torch.utils.data import DataLoader
from data.config import PipelineConfig
from data.dataset import load_jsonl, split_dataset
from data.curriculum_dataset import FIMDataset, HybridReplayDataset


def _collate_fn(batch: list[dict]) -> dict:
    """Collate list of FIM sample dicts into a batched dict with list values."""
    keys = ["fim_text", "prefix", "suffix", "middle", "n_lines"]
    return {k: [item[k] for item in batch] for k in keys}


class CurriculumDataModule(pl.LightningDataModule):
    def __init__(self, config: PipelineConfig, batch_size: int = 8):
        super().__init__()
        self.config = config
        self.batch_size = batch_size
        self.train_records = []
        self.val_records = []
        self.test_records = []

    def setup(self, stage: Optional[str] = None):
        records = load_jsonl(self.config.jsonl_path)
        self.train_records, self.val_records, self.test_records = split_dataset(
            records,
            train=self.config.train_ratio,
            val=self.config.val_ratio,
            test=self.config.test_ratio,
            seed=self.config.seed,
        )

    def train_dataloader(self, curriculum_stage: int = 1) -> DataLoader:
        current_cfg = self.config.stages[curriculum_stage - 1]
        current_ds = FIMDataset(self.train_records, current_cfg, seed=self.config.seed)
        prior_ds_list = [
            FIMDataset(self.train_records, self.config.stages[i], seed=self.config.seed)
            for i in range(curriculum_stage - 1)
        ]
        hybrid_ds = HybridReplayDataset(current_ds, prior_ds_list, replay_ratio=0.25)
        return DataLoader(hybrid_ds, batch_size=self.batch_size, shuffle=False, collate_fn=_collate_fn)

    def val_dataloader(self, curriculum_stage: int = 1) -> DataLoader:
        current_cfg = self.config.stages[curriculum_stage - 1]
        val_ds = FIMDataset(self.val_records, current_cfg, seed=self.config.seed)
        return DataLoader(val_ds, batch_size=self.batch_size, shuffle=False, collate_fn=_collate_fn)

    def test_dataloader(self) -> DataLoader:
        last_cfg = self.config.stages[-1]
        test_ds = FIMDataset(self.test_records, last_cfg, seed=self.config.seed)
        return DataLoader(test_ds, batch_size=self.batch_size, shuffle=False, collate_fn=_collate_fn)
```
