  <action>
Create `data/__init__.py` as an empty file.

Create `data/config.py` with two dataclasses using Python's `dataclasses` module (no external dependencies):

```python
from dataclasses import dataclass, field
from typing import List

@dataclass
class StageConfig:
    stage: int
    min_lines: int
    max_lines: int

@dataclass
class PipelineConfig:
    jsonl_path: str
    train_ratio: float = 0.8
    val_ratio: float = 0.1
    test_ratio: float = 0.1
    seed: int = 42
    stages: List[StageConfig] = field(default_factory=list)
```

Create `data/dataset.py`:
- `load_jsonl(path: str) -> list[dict]`: open file, iterate lines, skip blank lines, parse each with `json.loads`, return list. Raise `FileNotFoundError(f"JSONL file not found: {path}")` if file missing.
- `split_dataset(records: list[dict], train: float = 0.8, val: float = 0.1, test: float = 0.1, seed: int = 42) -> tuple[list, list, list]`:
  - Validate ratios sum to 1.0 (tolerance 1e-6); raise `ValueError("Ratios must sum to 1.0")` otherwise
  - Shuffle a copy using `random.Random(seed).shuffle(shuffled)` — do NOT mutate `records`
  - Compute `n_train = int(len * train)`, `n_val = int(len * val)`, remainder goes to test
  - Slice: `train_list = shuffled[:n_train]`, `val_list = shuffled[n_train:n_train+n_val]`, `test_list = shuffled[n_train+n_val:]`
  - Return `(train_list, val_list, test_list)`

Create `tests/__init__.py` as an empty file.

- `test_split_original_not_mutated`: original list unchanged after split

Use `pytest` — no mocking frameworks needed.
