FIM_BEGIN  = "<ÔΩúfim‚ñÅbeginÔΩú>"   # U+FF5C + U+FIM special chars ‚Äî copy from here
FIM_HOLE   = "<ÔΩúfim‚ñÅholeÔΩú>"
FIM_END    = "<ÔΩúfim‚ñÅendÔΩú>"
EOT        = "<|EOT|>"

PSM full format (training target):
  f"{FIM_BEGIN}{prefix}{FIM_HOLE}{suffix}{FIM_END}{middle}{EOT}"

Where:
  prefix = all lines BEFORE the gap, joined with "\n", trailing newline preserved
  suffix = all lines AFTER the gap, joined with "\n", leading newline preserved
  middle = the N masked lines joined with "\n" (this is the training target)
</interfaces>

  <behavior>
    Exclusion rules for `select_gap_lines`:
    - A line is INELIGIBLE if ANY of the following is true:
      1. `line.strip() == ""` (empty or whitespace-only)
      2. `line.strip().startswith("#")` (comment-only line)
      3. `re.match(r"^\s*(def |class )", line)` (function or class definition)
    - A line is eligible if none of the above apply
    - `select_gap_lines(lines: list[str], n: int, seed: int) -> tuple[int, int]`:
      - Find all maximal runs of N or more consecutive eligible lines
      - From those runs, pick a random starting position using `random.Random(seed)`
      - Return `(start_idx, end_idx)` where `end_idx = start_idx + n` (exclusive)
      - If no run of N consecutive eligible lines exists, raise `GapError(f"No valid gap of {n} lines found")`
    - `format_psm(lines: list[str], start: int, end: int) -> dict`:
      - Returns dict with keys: `"fim_text"` (full PSM string), `"prefix"`, `"suffix"`, `"middle"`
      - `prefix = "\n".join(lines[:start])` ‚Äî everything before the gap
      - `suffix = "\n".join(lines[end:])` ‚Äî everything after the gap
      - `middle = "\n".join(lines[start:end])` ‚Äî the masked lines (training target)
      - `fim_text = f"{FIM_BEGIN}{prefix}{FIM_HOLE}{suffix}{FIM_END}{middle}{EOT}"`
    - `create_fim_sample(code: str, n: int, seed: int) -> dict`:
      - Splits code on `"\n"` (preserving empty lines as empty strings for index alignment)
      - Calls `select_gap_lines`, then `format_psm`
      - Returns the dict from `format_psm` plus `"n_lines": n`

    Test cases for `tests/test_fim.py`:
    - `test_psm_token_structure`: given a simple 5-line code string, verify `fim_text` starts with `<ÔΩúfim‚ñÅbeginÔΩú>` and contains `<ÔΩúfim‚ñÅholeÔΩú>` and `<ÔΩúfim‚ñÅendÔΩú>` in that order, and ends with `<|EOT|>`
    - `test_middle_not_in_prefix_or_suffix`: the masked lines do NOT appear in prefix or suffix
    - `test_prefix_plus_middle_plus_suffix_reconstructs_code`: `prefix + "\n" + middle + "\n" + suffix == original_code` (modulo edge whitespace) ‚Äî confirms no lines are lost
    - `test_excludes_def_lines`: a code with only `def foo():` eligible lines raises `GapError`
    - `test_excludes_class_lines`: a code with only `class Foo:` eligible lines raises `GapError`
    - `test_excludes_empty_lines`: empty lines not selected into gap
    - `test_excludes_comment_lines`: comment-only lines (`# comment`) not selected
    - `test_deterministic_same_seed`: `create_fim_sample(code, n=1, seed=7)` called twice returns identical `fim_text`
    - `test_different_seeds_may_differ`: two seeds on code with multiple valid positions produce different start indices (use code with 10+ eligible lines)
    - `test_n_lines_respected`: returned `middle` has exactly `n` lines when `n=2`
    - `test_no_valid_gap_raises`: single-line code with only `def foo(): pass` raises `GapError`

2. Define `class GapError(ValueError): pass` ‚Äî raised when no valid N-consecutive-line gap exists.

3. Implement `_is_ineligible(line: str) -> bool`:
   - Returns True if `line.strip() == ""`
   - Returns True if `line.strip().startswith("#")`
   - Returns True if `re.match(r"^\s*(def |class )", line)` matches
   - Returns False otherwise

4. Implement `select_gap_lines(lines: list[str], n: int, seed: int) -> tuple[int, int]`:
   - Build list of eligible indices: `[i for i, l in enumerate(lines) if not _is_ineligible(l)]`
   - Find all starting positions `i` such that `eligible_indices` contains `n` consecutive values starting at `i` (i.e., `eligible_indices[j], eligible_indices[j]+1, ..., eligible_indices[j]+n-1` are all present)
   - Algorithm: scan eligible_indices for runs of consecutive integers of length >= n. For each run of length >= n, add all valid start positions (any window of n within that run). Collect as `valid_starts`.
   - If `valid_starts` is empty, raise `GapError(f"No valid gap of {n} consecutive eligible lines found in {len(lines)}-line code")`
   - Pick `start = random.Random(seed).choice(valid_starts)`
   - Return `(start, start + n)`

5. Implement `format_psm(lines: list[str], start: int, end: int) -> dict`:
   - `prefix = "\n".join(lines[:start])`
   - `suffix = "\n".join(lines[end:])`
   - `middle = "\n".join(lines[start:end])`
   - `fim_text = f"{FIM_BEGIN}{prefix}{FIM_HOLE}{suffix}{FIM_END}{middle}{EOT}"`
   - Return `{"fim_text": fim_text, "prefix": prefix, "suffix": suffix, "middle": middle}`

6. Implement `create_fim_sample(code: str, n: int, seed: int = 42) -> dict`:
   - `lines = code.split("\n")`
   - `start, end = select_gap_lines(lines, n, seed)`
   - `result = format_psm(lines, start, end)`
   - `result["n_lines"] = n`
   - Return result
