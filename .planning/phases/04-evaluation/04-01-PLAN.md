    ```python
    def edit_similarity(predicted: str, expected: str) -> float:
    ```

    ```python
    def run_eval(model, tokenizer, dataloader, ground_truths: list[str], max_new_tokens: int) -> dict:
    ```
    The run_eval function:
    - Iterates over dataloader batches
    - For each batch, calls model.model.generate(input_ids=batch["input_ids"], attention_mask=batch["attention_mask"], do_sample=False, num_beams=1, max_new_tokens=max_new_tokens)
    - Decodes generated tokens with tokenizer.decode(ids, skip_special_tokens=False)
    - Extracts the middle portion: splits on FIM_END token, takes the text after FIM_END and before EOT. If FIM_END not found, uses empty string.
    - Compares each predicted middle against corresponding ground_truth entry
    - Returns {"exact_match": mean_em, "edit_sim": mean_es}

    NOTE: The dataloader used for eval produces tokenized batches (via TokenizedCollator). The ground_truths list is assembled separately by the caller from raw FIM samples. run_eval does NOT extract ground truth from the dataloader — it receives it as a parameter.

    Import FIM_END and EOT from data.fim.
